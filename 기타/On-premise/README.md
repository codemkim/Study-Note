* 기업의 서버를 클라우드 같은 원격 환경에서 운영하는 방식이 아닌, <br>
  자체적으로 보유한 전산실 서버에 직접 설치해 운영하는 방식

## Cloud 시스템의 단점
* 클라우드 시스템을 이용하게되면 클라우드 시스템을 익혀야하는 학습 비용,<br>
  제한된 접근 권한으로 인한 문제 대응의 어려움, 온프레미스보다 복잡한 인프라 체계,<br>
  지속적으로 사용시 온프레미스에 비해 많이 발생할 수 있는 비용
